# ⏱️ Stopwatch

A clean, minimal stopwatch web app with an analog clock face, lap tracking, and keyboard shortcuts — built with pure HTML, CSS, and JavaScript.

## ✨ Features

- **Analog clock face** — SVG hands that move in real time
- **Digital display** — centisecond precision (`MM:SS.cs`)
- **Start / Pause / Resume / Reset** controls
- **Lap recording** — tracks split times and highlights the best 🟢 and worst 🔴 laps
- **Keyboard shortcuts** — `Space` to start/pause, `L` to lap, `R` to reset
- **Status indicator** — animated pill showing Ready / Running / Paused state
- **No dependencies** — pure vanilla HTML, CSS, and JavaScript

## 📁 Project Structure

```
stopwatch/
├── index.html   # App markup and structure
├── style.css    # All styling (dark theme)
└── script.js    # Stopwatch logic and DOM interactions
```

## 🚀 Getting Started

No build tools or installs needed. Just open `index.html` in any browser:

```bash
# Clone the repo
git clone https://github.com/khushi-gandhi1/stopwatch.git

# Open in browser
open stopwatch/index.html
```

Or simply drag `index.html` into your browser window.

## ⌨️ Keyboard Shortcuts

| Key     | Action        |
